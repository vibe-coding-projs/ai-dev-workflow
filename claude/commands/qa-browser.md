Run interactive QA on the preview environment using your local Chrome browser:

1. Get the preview URL from the Jira ticket comments
2. Open the preview URL in the local Chrome browser via chrome-local-mcp
3. Check each acceptance criterion from the Jira ticket visually
4. Check layout, responsiveness, and visual appearance interactively
5. Take screenshots of each key screen/state
6. Report what passed and what failed
7. Post a QA summary comment on the Jira ticket with results and screenshots
